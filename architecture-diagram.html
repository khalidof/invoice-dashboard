Invoice Upload

Claude Tool Use Initialization

n8n Code node calls Anthropic API directly with tool definitions. No LangChain - pure Claude tool calling.

3

Document Analysis

Claude Vision analyzes the document. Uses MCP tools iteratively to extract and validate each field with confidence scoring.

Vision API MCP Tools
4

Cross-Validation

Agent uses Supabase MCP to check for duplicates, validate vendor exists, and verify extracted amounts against historical data.

Supabase MCP SQL
5

Confidence Evaluation

If confidence < 85%, agent re-extracts uncertain fields. Low confidence items are flagged for human review.

Validation Retry Logic
6

Response & Storage

Extracted data returned to frontend. Frontend stores in Supabase with file URL, extracted fields, and confidence scores.
